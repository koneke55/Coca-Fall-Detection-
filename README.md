# Coca Fall Detection

This project focuses on detecting human falls using sensor data and computer vision techniques. Below is a list of publicly available fall detection datasets that are commonly used for research and benchmarking.

## 📂 Available Datasets

- **UR Fall Detection Dataset** – acceleration and gyroscope data from wearable devices.  The dataset includes
  both falls and activities of daily living.  
- **SisFall** – large-scale inertial sensor dataset collected from smartphones and wearable IMUs.  
- **Le2i Fall Detection Dataset** – video sequences captured from multiple cameras, includes elderly people
  performing falls and normal actions.  
- **Multiple Cameras Fall Dataset (MCFD)** – multi-view RGB and depth recordings of staged falls.  
- **NTU RGB+D** – though not solely for falls, contains fall actions within a broader action recognition
  collection, often used for skeleton-based detection.

> **Tagging tip:** when organizing data, tag files or records by source (e.g. `#URFalls`), modality
> (`#IMU`, `#Video`, `#Depth`), and scenario (`#Elderly`, `#Simulated`).

The above datasets provide a starting point for training and validating fall-detection algorithms. Feel
free to extend this list with new sources or custom collections.
